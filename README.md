# | Sistema Bancário em Python

Projeto desenvolvido em Python simulando um sistema bancário simples via terminal.


## | Sobre

Este projeto simula operações básicas de um sistema bancário, permitindo ao usuário realizar ações como depósito, saque e visualização de dados do cartão.

O sistema foi desenvolvido com foco em lógica de programação, validação de dados e interação com o usuário via terminal.



## | Funcionalidades

- Autenticação com:
  - Últimos dígitos do cartão  
  - CPF  
  - Senha  

- Operações disponíveis:
  -  Depósito  
  -  Saque  
  -  Visualização do cartão  
  -  Encerramento do sistema  



## | Tecnologias utilizadas

- Python  



## | Conceitos aplicados

- Estruturas condicionais (`if/else`)  
- Estruturas de repetição (`while`)  
- Validação de entrada de dados  
- Tratamento de erros (valores inválidos)  
- Lógica de controle de saldo  



## | Estrutura do Projeto

- `Banco.ipynb` → Código principal do sistema bancário

## | Destaque

O projeto realiza validações importantes, como:

- Impedir saque com saldo insuficiente  
- Garantir que valores digitados sejam numéricos  
- Validar valores positivos para depósito  



## | Observação

Este é um projeto de estudo com foco em aprendizado de lógica e não possui integração com banco de dados real.
